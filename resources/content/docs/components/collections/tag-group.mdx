## Basic
A tag group is a focusable collection of labels, categories, keywords, filters, or other elements, equipped with keyboard navigation, selection, and removal functionalities.

## Variants
Alter the appearance of the tag group by utilizing the `intent` prop. Currently, only the default blue color aligns with the selected variant, sourced from the badge component. Explore additional variants in the [badge documentation](/design-system/statuses/badge).

```
<TagGroup intent="info"/>
```
