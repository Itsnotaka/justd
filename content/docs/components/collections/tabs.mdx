## Disabled
Tabs can be disabled by setting the `isDisabled` prop to `true`.
<How isCenter={false} toUse="navigation/tabs-disabled-demo" />

You can also disable individual tabs by setting the `isDisabled` prop to `true`, like this:
```tsx
<Tab isDisabled id="c">Contact</Tab>
<Tab isDisabled id="a">About Us</Tab>
```
