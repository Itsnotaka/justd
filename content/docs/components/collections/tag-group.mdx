## Basic
A tag group is a focusable list of labels, categories, keywords, filters, or other items, supporting keyboard navigation, selection, and removal.

## Variants
You can switch up the tag group intent by throwing in the `intent` prop. But right now, only the default blue matches the selected color. The color comes from the badge component. If you want, check out the available variants in the [badge docs](/design-system/statuses/badge).

```
<TagGroup intent="info"/>
```
