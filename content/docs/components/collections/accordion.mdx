## Default Expanded
You can set the default expanded keys by passing an array of keys to `defaultExpandedKeys` prop.
<How isCenter={false} toUse="controls/accordion-default-expand-demo" />

## Hide Indicator
You can hide the arrow key indicator by passing `hideIndicator` prop.
<How isCenter={false} toUse="controls/accordion-hide-indicator-demo" />

## Disabled
Flip the switch on the accordion by chucking in a `disabledKeys` prop. Peep this example: I put the brakes on keys `1`, `2`, and `4`. Once they're locked down, you can't pop 'em open or even mess with 'em.
