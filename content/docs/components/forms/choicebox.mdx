## Controlled
This component can be controlled or uncontrolled. To make it controlled, pass the `selectedKeys` prop and use the `onSelectionChange` handler to update it.
<How isCenter={false} withNoPadding={true} toUse="forms/choicebox-controlled-demo" />

## Single Selection
By default, the Choice Box is multi-selectable. To make it single-select, pass the `selectionMode` prop.

<How isCenter={false} withNoPadding={true} toUse="forms/choicebox-single-selection-demo" />

## Columns and Gap
By default, the Choicebox is displayed 2 columns, and the gap between items is 16px. You can change these values by passing the `columns` and `gap` props.

<How isCenter={false} withNoPadding={true} toUse="forms/choicebox-columns-demo" />

## Disabled
You can disable the entire Choice Box by passing the `isDisabled` prop to the item.

<How isCenter={false} withNoPadding={true} toUse="forms/choicebox-disabled-demo" />

You can also disable individual items by passing an array of keys to the `disabledKeys` prop.

```tsx
<Choicebox disabledKeys={['sm', 'lg']} />
```
