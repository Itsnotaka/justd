## Alert Dialog
When we chat about **alerts** dialog, peep this: you gotta nail two things. First, lock it down so the modal ain't closin' when you tap outside. Second, make that close button ghost unless it's handed over. Simple fix, just slide the `isDismissable` prop to the modal, and you’re set.
<How toUse="overlays/alert-dialog-demo" />
So you might be wonderin', what's the move when we ain't rollin' with the `Modal`? Say you're tryna handle that modal vibe, straight up, you don't even need the `Modal` itself. Nah, switch gears and hit up the `ModalOverlay`. Just toss the `isDismissable` prop its way, and you're golden.
```
<ModalOverlay isDismissable={false}/>
```

## Triggered By Menu
You can also trigger the modal by clicking on a menu item.
<How toUse="overlays/modal-triggered-by-menu-demo" />
But it would be better if we can extract the modal to a component.

```
interface ModalActionProps {
  title: string
  description: string
  confirmText: string
  action: () => void
  closeModal: () => void
  loading: boolean
  isOpen: boolean
}

export function ModalAction(props: ModalActionProps) {
  return (
    <ModalOverlay isOpen={props.isOpen} onOpenChange={props.closeModal}>
      <ModalContent>
        <ModalHeader>
          <ModalTitle>{props.title}</ModalTitle>
          <ModalDescription>{props.description}</ModalDescription>
        </ModalHeader>
        <ModalFooter>
          <ModalClose>Cancel</ModalClose>
          <Button className="min-w-24" isDisabled={props.loading} onPress={props.action}>
            {props.loading ? <LoadingDots className={'bg-fg'} /> : props.confirmText}
          </Button>
        </ModalFooter>
      </ModalContent>
    </ModalOverlay>
  )
}
```

Then you can use it like this.

```
<ModalAction {...actionType(state)} closeModal={closeModal} loading={loading} isOpen={!!state} />
```

With that, now we can modify the `actionType` function to return the initial state.

```tsx
const actionType = (t: string | null) => {
  let initialsState = {
    title: '',
    description: '',
    confirmText: '',
    action: () => {}
  }

  switch (t) {
    case 'delete': ...
    case 'ban': ...
    case 'restore': ...
    default:
      return initialsState
  }
}
```
